    <h1>FamilyHub</h1>

    <p>
      FamilyHub represents a family planner web application designed to help households stay organized by managing events, chores, and daily responsibilities in one centralized place. Instead of relying on multiple apps, calendars, or group messages, this project provides a simple and mobile-friendly solution where family members can quickly view schedules, assign tasks, and stay connected. The site includes a homepage that highlights chores and responsibilities, along with a calendar page that displays a monthly view of events and allows users to add new activities for the family.
    </p>

    <h2>Purpose</h2>

    <p>
      As someone balancing work, school, and family life, I have experienced how difficult it can be to keep everyone aligned on schedules and responsibilities. Important tasks and events can easily get lost between texts, conversations, and different apps. I built this project to create a simple and clear system that makes it easier for families to stay organized without added complexity. FamilyHub is designed for busy households who want a quick way to track chores, manage events, and maintain visibility across all family members, especially when schedules do not always align.
    </p>

    <p><strong>JavaScript</strong> - I used JavaScript to add interactivity, including creating and deleting events and dynamically updating the interface.</p>
